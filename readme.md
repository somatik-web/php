# SomatikSoundSystem PHP Version site

## Usage

### Running:
```shell script
docker-compose up
```
or (in case of rebuild)
```shell script
docker-compose up --build
```

### Restart single service:
```shell script
docker-compose restart <service name>
```

### Exec Composer command
```shell script
docker-compose run composer <command>
```
